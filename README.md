


## 🎮 How to Play
- Use the **Arrow Keys** (↑ ↓ ← →) or **W, A, S, D** to move Pacman.
- Eat all the **white dots** to score points.
- Avoid **ghosts** — colliding with them will cost you a life.
- You have **3 lives**. Game over when all lives are lost.

## ⚡ Features
- Animated **Pacman** movement with directional images.
- Multiple **ghosts** with simple AI movement.
- Score and life counter displayed.
- Responsive game board built with JavaScript.


## 💾 How to Download & Play the game
1. Click the green Code button at the top of this repository.
2. Select Download ZIP
3. Extract the ZIP folder to your computer.
4. Open index.html in a web browser to play locally.
